##  English

### Description

A minimalist, dark-themed typing speed test inspired by Monkeytype. Test your typing skills with real-time statistics including WPM (words per minute), accuracy percentage, and error count. The timer starts automatically when you begin typing.

### Features

- 憋 Multiple duration modes (15s, 30s, 60s)
-  Real-time WPM and accuracy tracking
-  Leaderboard and history saved locally
-  Dark mode interface
-  Export/Import results as JSON

##  Statistics Explained

| Stat | Formula |
|------|---------|
| **WPM** | `(correctChars / 5) / minutes` |
| **Accuracy** | `(correctChars / totalTyped)  100` |
| **Errors** | Characters that didn't match the target |
